## Run Locally

Clone the project

```bash
  git clone https://github.com/Asafyaco/Rick-and-Morty
```


Install dependencies

```bash!
  npm install
```

Start the server

```bash
  npm start
```

## Tech Stack :computer:

**Client:** JavaScript, Scss, React

**Deploy:** Heroku
